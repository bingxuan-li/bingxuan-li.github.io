ROMA on Mitsuba

This project explores the further applications of ROMA in rendering, specifically implemented on Mitsuba 0.6. Under the guidance of Prof. Lingqi Yan from UC Santa Barbara, we primaryly focused on enhancing the performance of ray tracing using ROMA, a novel acceleration method.
BVH Loader

A simple BVH (Biovision Hierarchy) loader written in Python. BVH is a file format originally developed by Biovision for storing motion capture data. BVH files are commonly used in computer graphics and animation to store skeletal motion data, making them a valuable asset in various applications.
ParticleSun

The project leverages particles to create a visual representation of a sun's corona. It utilizes OpenGL as its backend. There is also a moon follows Newton's Law to simulate a realistic orbit, and it's rendered using the Blinn-Phong shading model.

RSA Hacker

In this project, we delve into the dark side, aiming to crack RSA keys generated in various vulnerable ways. The implemented algorithms focus on classical techniques to compromise the security of RSA keys. Here is a project report.
FightTheLandlord2 AI

This project is an AI implementation for the card game FightTheLandlord2. FightTheLandlord2 is a complex and strategic card game, and the AI aims to provide a challenging opponent for players on the Botzone platform.
NoGo

This project is a game implementation of NoGo, written in C and featuring a simple GUI created using EasyX. NoGo is a two-player strategic board game, and you can find more information about the game itself here.
